O Summ-Tool é uma ferramenta para sumarização de textos em português utilizando abordagem extrativa. Nesta ferramenta estão implementadas diversas opções de pré-processamento, como segmentação e normalização de textos, POS (Part-Of-Speech Tagger) e NER (Named entity recognition). Também são disponibilizadas diversas opções de features para análise do texto. Como mecanismo de tomada de decisão é possível utilizar lógica fuzzy, algoritmos genéticos e em breve redes neurais.

O Summ-Tool foi implementado utilizando Java. Bibliotecas como Apache OpenNLP, Java ROUGE 2.0 e jFuzzyLogic foram utilizadas. Em breve disponibilizarei o link para o repositório do código fonte do projeto.
