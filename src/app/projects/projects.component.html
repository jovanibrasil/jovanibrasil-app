Aqui estão listados os projetos que venho desenvolvendo nos últimos meses. Para tal desenvolvimento, estão sendo utilizadas diversas tecnologias com alta demanda na indústria de desenvolvimento de software. Devido a limitações dos servidores utilizados, qualquer aplicação minha que esteja disponível ao público pode sobrer instabilidades ou até mesmo ficar indisponível.

A ideia é com o tempo ir especificando melhor os detalhes de cada um dos projetos e compartilhar esse conhecimento por meio do meu blog e do meu futuro canal no Youtube. Também em um futuro próximo produzirei alguns cursos explicando o passo a passo para desenvolvimento das minhas aplicações.

Sumarizador de textos (Java 8 + Lógica Fuzzy + Algoritmos Genéticos + PLN)

Ferramenta para sumarização de textos em português utilizando abordagem extrativa. Nesta ferramenta estão implementadas diversas opções de pré-processamento e features para análise do texto. Como mecanismo de tomada de decisão é possível utilizar lógica fuzzy e em breve algoritmos genéticos e redes neurais.

Implementado utilizando Java 8, utiliza bibliotecas como Apache OpenNLP, Java ROUGE 2.0 e jFuzzyLogic.

Aqui está o link para o repositório do código fonte do projeto.

Search API (Solr + Spark)

API de busca implementada utilizando Spark e Solr. Atualmente está sendo utilizada para buscas no meu blog. Note que ela está em desenvolvimento é resultado de um processo de aprendizagem, então qualquer problema certamente será corrigido no futuro.

A API Spark e a instância Solr rodam em containers distintos e o Solr está configurado para integrar com o MySQL. O deploy correto depende de uma instância do Vault devidamente configurada e acessada via Jenkins, além de um banco em uma instância MySQL.

Sniffer (Java 8 + C)

Neste projeto o objetivo é aprender mais sobre a integração de Java com linguagens de mais baixo nível, como C. O recurso que permite integração e que foi utilizado aqui é o JNI (Java Native Interface). Um Sniffer implementado em C (que utiliza basicamente RAW sockets) está sendo integrado com uma GUI (Graphical User Interface). A ideia inicial é realizar uma simples integração e com o tempo novas features deverão ser consideradas no projeto.
