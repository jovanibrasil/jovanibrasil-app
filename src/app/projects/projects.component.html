Aqui estão listados os projetos que venho desenvolvendo nos últimos meses. No desenvolvimento dos meus projetos estão sendo utilizadas diversas tecnologias com alta demanda na indústria de desenvolvimento de software.

No que se refere a projetos web, que é o meu foco atual, no frontend estou majoritariamente utilizando Angular e Bootstrap, e recentemente comecei um projeto com React. Já no backend estou utilizado JAVA com Spring Framework na maioria dos projetos, porém também recentemente iniciei um projeto utilizando Node e Express. No tocante a banco de dados, são utilizados os bancos MySQL e MongoDB. Nos projetos desktop tenho utilizado Python e Java SE, Python quando se refere a análise de dados, e Java para os demais tipos de aplicações. Utilizo o Github como repositório dos projetos e um servidor na AWS (configurado do zero) para rodar os projetos com com toda uma infra de containers usando Docker. Para integração e deploy das aplicações com um bom grau de automação, tenho utilizado Jenkins.

A ideia é com o tempo ir especificando melhor os detalhes de cada um dos projetos e compartilhar esse conhecimento por meio do meu blog e do meu futuro canal no Youtube.

Lembre-se, qualquer dúvida, sugestão, conselho ou reclamação, fique a vontade para me enviar um email.
