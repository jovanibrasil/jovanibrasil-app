    Sobre as páginas Home, Projects e CV

    As páginas Home, Projects e CV do frontend são implementadas sem a utilização de bibliotecas ou frameworks, utilizando apenas recursos básicos do desenvolvimento web, como html, css e javascript. Para o seu backend, foi utilizado node runtime para rodar um servidor em javascript responsável por servir as páginas web. Note que em Blog ainda está vazio, pois ainda nem foram definidas as tecnologias que serão empregadas.

    A partir da página Projects, é possível ser redirecionado para projetos específicos que utilizam diferentes tipos de tecnologias no frontend e no backend. Você pode verificar da seção de cada projeto, quais foram as tecnologias utilizadas, assim como o porquê o como da escolha de cada tecnologia.
