  • Este Web-site

    • O objetivo desta página é aplicar e demonstrar os conhecimentos obtidos nos meus estudos em diversas tecnologias. Por esse motivo, foram utilizadas diferentes tecnologias nos diferentes projetos que são ou serão demonstrados aqui.

    Sobre as páginas Home, Projects e CV

    As páginas Home, Projects e CV do frontend são implementadas sem a utilização de bibliotecas ou frameworks, utilizando apenas recursos básicos do desenvolvimento web, como html, css e javascript. Para o seu backend, foi utilizado o node runtime para rodar um servidor em javascript responsável por servir as páginas web. Note que em Blog ainda está vazio, pois ainda estão sendo definidas as tecnologias que serão utilizadas em sua implementação, que deverá ocorrer muito em breve.

    A partir da página Projects, é possível ser redirecionado para projetos específicos que utilizam diferentes tipos de tecnologias no frontend e no backend. Você pode verificar da seção de cada projeto, quais foram as tecnologias utilizadas, assim como o porquê o como da escolha de cada tecnologia.

    Infraestrutura

    Atualmente esta página web e minhas soluções estão hospedadas em um servidor, utilizando diversos Docker containers para conciliar de forma escalável e confiável os diversos tipos de tecnologias. Para orquestração está sendo utilizado Docker compose, devido a simplicidade do projeto. Um servidor NGINX é utilizado para implementação de um proxy reverso no sistema.

      • Loja Web Java

        A ideia deste projeto é desenvolver uma loja Web em Java. O objetivo é, ao fim do projeto, ter as funcionalidas básicas de uma loja web qualquer, como listagens de produtos, carrinho de compras, login de usuários e gerênciamento de produtos do site.
